# InforMARL

Run the selected `env_id` from `Config.py`:

```bash
python main.py
```

Optional `env_id` override:

```bash
python main.py formation
python main.py line
python main.py graph_navigation
```

Only the requested GraphMPE scenarios are included:

- `simple_graph_formation`
- `line_graph`
- `navigation_graph`
